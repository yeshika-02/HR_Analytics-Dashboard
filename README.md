# HR Analytics Dashboard (Power BI)

## Project Overview

This project is an interactive HR Analytics Dashboard built using Power BI. It helps analyze employee demographics, attrition trends, compensation patterns, and performance metrics.

## Dashboard Pages

### Executive Summary

* Total Employees
* Active Employees
* Attrition Count
* Attrition Rate

### Attrition Analysis

* Attrition by Job Role
* Attrition by Age Group
* Attrition by Salary Band
* Attrition by Travel Frequency
* Attrition by Overtime

### Employee Demographics

* Gender Distribution
* Age Group Analysis
* Education Level
* Department Distribution
* Marital Status

### Compensation & Performance Analysis

* Average Salary by Department
* Average Salary by Job Role
* Salary Hike Analysis
* Job Satisfaction
* Promotion Analysis

## Tools Used

* Power BI
* DAX
* CSV Dataset
* Data Visualization

## Key Insights

* Attrition Rate: 16.08%
* Employees aged 26–35 show the highest attrition.
* Low salary bands contribute most to attrition.
* Research & Development has the largest workforce.
* Bachelor's degree holders form the largest education group.
