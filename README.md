# HR Analytics Dashboard

## Project Overview
Interactive HR Analytics Dashboard built using Power BI.

## Dashboard Pages
- Executive Summary
- Attrition Analysis
- Employee Demographics
- Compensation & Performance Analysis

## Tools Used
- Power BI
- DAX
- CSV Dataset

## Key Insights
- Attrition Rate: 16.08%
- Employees aged 26–35 show highest attrition.
- Low salary bands have the highest turnover.
- Research & Development has the largest workforce.
